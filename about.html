MarkdownBrowser

Local markdown file explorer for Chrome

Version 1.0.0

About

Markdown Browser is a free Chrome extension that lets you read, browse, and search your local .md files directly in your browser — with no uploads, no accounts, and no internet connection required.

Your files are accessed using Chrome's File System Access API, which reads your local folder with your explicit permission. Nothing ever leaves your machine.

Features

Sidebar file tree Collapsible folders, live filter, recent files history
Hot reload Auto-refreshes when you save in your editor
Full-text search Search filenames and content with ⌘K
Syntax highlighting JS, TS, Python, Go, Rust, SQL, CSS, HTML, Bash, JSON
Table of contents Auto-generated with scroll-spy
Light & dark mode Toggle in the navbar, preference saved
GitHub Flavored Markdown Tables, task lists, front matter, code blocks
100% offline No network requests, no data collection

Privacy

This extension does not collect any personal data, does not make any network requests, and does not communicate with any external servers. All processing happens locally on your device.
