MarkdownBrowser

A local markdown reader & editor — fast, private, offline.

About

Markdown Browser lets you read, write, and search your local .md files directly in Chrome — no uploads, no cloud, no internet required. Your files stay on your machine.

Built with the File System Access API so Chrome accesses your folder directly with your explicit permission.

How to use

1
Open a folder

Click Open Folder in the sidebar and select any local directory. All .md, .mdx, and .markdown files will appear in the file tree.

2
Browse & read

Click any file to open it. Use the Contents tab to jump to headings, and Recent to revisit past files. Use ⌘[ / ⌘] to navigate between files.

3
Edit & save

Click Edit in the file breadcrumb (or press ⌘E) to open the in-browser editor. Press ⌘S to save. Use ⌘F for find & replace.

4
Create & delete

Click the + button in the sidebar to create a new file. Hover any file to reveal the delete button. All file changes require your explicit permission.

5
Search

Click the search bar in the header (or press ⌘K) to search filenames and content across your entire folder instantly.

6
Export & hot reload

Click PDF in the breadcrumb to export a clean print. Toggle Live in the header to auto-refresh when you save in an external editor.

Keyboard shortcuts

⌘K Search files and content ⌘E Enter / exit edit mode ⌘S Save file (edit mode) ⌘F Find & replace (edit mode) ⌘[ Previous file ⌘] Next file ⌘B Toggle sidebar ESC Close palette, settings, or find bar

Features

  • In-browser editor — edit files directly, save with ⌘S
  • Create & delete files — manage your markdown folder without leaving the app
  • Find & Replace — search and replace within the editor
  • Export to PDF — clean, print-ready output
  • Full GitHub Flavored Markdown — tables, task lists, front matter, code blocks
  • Syntax highlighting for JS, TS, Python, Go, Rust, SQL, CSS, HTML, Bash, JSON
  • Hot reload — live preview as you edit in any external editor
  • Full-text search across all files in the folder
  • Table of contents with scroll-spy highlighting
  • Resizable sidebar, font size & reading width controls
  • Light & dark mode — toggle in the header
  • 100% offline & private — no data ever leaves your machine
